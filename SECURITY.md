# Security Policy

## Supported Versions

Only the latest released version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.7.x   | :white_check_mark: |
| < 1.7.0 | :x:                |

## Reporting a Vulnerability

Please **do not open a public issue** for security vulnerabilities (e.g.
credential/token leakage, unsafe handling of secrets, injection issues, or
anything that could expose a user's Bosch Smart Home account or camera
stream to unauthorized access).

Report privately instead, via:

* [GitHub Security Advisories](https://github.com/mosandlt/ioBroker.bosch-smart-home-camera/security/advisories/new) for this repository (preferred)
* A private message to the maintainer via their [GitHub profile](https://github.com/mosandlt)

You can expect an acknowledgement within a few days. If the vulnerability is
accepted, a fix will be released as soon as reasonably possible and you will
be credited in the release notes (unless you prefer to remain anonymous). If
declined, we'll explain why.
