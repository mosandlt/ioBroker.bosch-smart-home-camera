# Contributing

Thanks for considering a contribution!

## Getting started

```bash
git clone https://github.com/mosandlt/ioBroker.bosch-smart-home-camera.git
cd ioBroker.bosch-smart-home-camera
npm install
```

## Before opening a PR

```bash
npm run lint
npm test
```

Both must pass. Please add or update tests under `test/` for any behavior
change.

## Reporting bugs

Please open an issue using the bug report template and include:

* Your ioBroker and Node.js version
* Steps to reproduce
* Relevant logs (with any tokens/secrets redacted)

## Security issues

Please do **not** open a public issue for security vulnerabilities — see
[SECURITY.md](SECURITY.md).
